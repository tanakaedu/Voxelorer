# Voxelorer
体験入学などで拡張していくボクセルキャラクターの冒険譚ゲームのリポジトリーです

# 前提
- Unity2017.3.1

# 便利アセット
ステージを作成する時に、ProGridsがインポートされていると便利です。

- <i>Window</i>メニューから<i>Asset Store</i>を選択してアセットストアを開きます
- `ProGrids`を検索
- ダウンロードしてImport
- グリッドを`0.5`か`0.25`にすると、本プロジェクトでは便利です
